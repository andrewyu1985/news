# News Digest

[Пример готового дайджеста](https://www.facebook.com/alex.v.krol/posts/pfbid02oj14ZFeSvyrrpcNN8dBJoJ6YsegA4gSeqtsSdhVMjkAYZU15aFuRH7msPN3EuE8al)

Полуавтоматический пайплайн для создания авторского новостного дайджеста. Chrome-расширение собирает статьи с Perplexity AI, а Claude Cowork (или любая другая LLM) превращает их в авторские комментарии на русском языке и собирает готовый дайджест.

Промпты определяют тон, стиль и формат подачи. Хотите другую подачу — просто меняйте промпты под себя. Никаких ограничений: формат, язык, тон, длина — всё настраивается через `prompt.md`, `prompt_deep.md` и `assembly_prompt.md`.

## Как это работает

```
Вкладки Perplexity  →  Chrome-расширение  →  JSON-файл  →  Claude Cowork  →  Готовый дайджест
```

### Шаг 1. Сбор новостей (Chrome-расширение)

Расширение из папки `extension/` находит все открытые вкладки Perplexity AI (`/discover/...` и `/page/...`), извлекает заголовок и текст каждой статьи и сохраняет всё в JSON-файл.

**Установка расширения:**

1. Открыть `chrome://extensions/`
2. Включить «Режим разработчика» (Developer mode)
3. Нажать «Загрузить распакованное расширение» (Load unpacked)
4. Выбрать папку `extension/`

**Использование:**

1. Открыть нужные статьи на Perplexity AI в отдельных вкладках (рекомендуется 10-15)
2. Нажать иконку расширения в панели Chrome
3. Нажать «Собрать новости»
4. Расширение скачает файл `input_ДАТА.json` со всеми статьями

> Паттерн URL сейчас настроен на `perplexity.ai/discover/*` и `perplexity.ai/page/*`. При желании его можно изменить в `extension/popup.js` — регулярное выражение `PERPLEXITY_PATTERN` в начале файла.

### Шаг 2. Обработка новостей (Claude Cowork)

Скормить JSON-файл в Claude Cowork вместе с промптом `prompt.md`. Claude прочитает каждую статью и напишет по ней авторский комментарий — короткий, ироничный, на русском языке.

Альтернативный промпт `prompt_deep.md` создаёт более философские, метафоричные тексты с прогнозами.

### Шаг 3. Сборка дайджеста (Claude Cowork)

Скормить готовые комментарии в Claude Cowork вместе с промптом `assembly_prompt.md` и файлом `config.md`. Claude соберёт всё в единый пронумерованный дайджест с хэштегом, ссылками на оригиналы, упоминанием курса и дисклеймером.

## Структура проекта

```
.
├── extension/          # Chrome-расширение для сбора статей
│   ├── manifest.json
│   ├── popup.html
│   ├── popup.js
│   └── icon*.png
├── prompt.md           # Промпт для обработки новостей (ироничный тон)
├── prompt_deep.md      # Промпт для глубокой аналитики (философский тон)
├── assembly_prompt.md  # Промпт для сборки финального дайджеста
├── config.md           # Настройки: хэштег, текст курса, дисклеймер
├── input_*.json        # Собранные статьи (не в git)
└── output/             # Готовые дайджесты (не в git)
```

## Настройка

Файл `config.md` содержит:

- **Хэштег** — ставится в начало дайджеста
- **Упоминание курса** — вставляется в середину дайджеста между комментариями
- **Граница** — дисклеймер в конце (генерируется каждый раз)
- **Паттерн URL** — какие страницы Perplexity собирать
